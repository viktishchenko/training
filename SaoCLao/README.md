# SaoCLao

`SaoCLao` is a simple music app which uses [SoundCloud API](https://soundcloud.com/stream).
It allows you listen to the right music, wherever you are. 

# What can I do with it? #
- Stream music
- Like/unlike a song
- Register account
- Login
